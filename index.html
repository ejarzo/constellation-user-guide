Overview

Constellation is a MIDI sequencer/arpeggiator for Ableton Live. It allows you to compose melodies and harmonies by manipulating a polygon.

Installation

To get started you can simply drag the Constellation.amxd file into Ableton Live. Ideally you will also want to save it to your User Library. To do this right click on "User Library" in the Ableton sidebar and select "Show in Finder/Explorer". If you don't already have a folder there for Max Devices, you can create one and place Constellation.amxd inside.

Controls

Use the polygon editor to create your sequences. Edges represent notes, and the angles at each point determines the musical interval between notes.

Click on the keyboard to set the shape's starting note. This can also be controlled with MIDI input.

Editor

  • Click on a midpoint to create a new point.
  • Double Click a point to delete it.
  • Option Drag a point (up/down) to change that note's velocity.
  • Click Reset to clear all but two points
  • Click Random to generate a random shape
  • Click Reverse to reverse the direction of playback. This will recalculate the notes based on the new angles (instead of simply playing the notes in opposite order)
  • Click Grid to toggle a grid, useful for creating quantized rhythms

Saved Shapes

  • Click on one of the four empty slots to the right of the editor to save the current shape
  • Click again to recall that shape
  • Shift Click to remove a saved shape

Output Mode

  • Rhythm mode outputs a single note (useful from drums)
  • Melody mode outputs a melodic sequence

Tempo Sync

  • With Tempo Sync ON, one loop of the shape will equate to the selected number of bars
  • With Tempo Sync OFF, you can adjust the speed that the shape loops, independent of the tempo. This means that larger shapes will take longer to loop.

Key and Mode

  • Use the dropdown menus to select key and mode
  • Selecting Chromatic will make all notes available

MIDI Thru

  • With MIDI Thru ON, incoming MIDI messages will NOT affect the shape's starting note. This allows you to jam over the top of the loop. This also means that two Constellation plugins on the same track will play together in parallel.
  • With MIDI Thru OFF, incoming MIDI messages WILL control the shape's starting note. This means that you can stack Constellation plugins and have the first one control the second.
